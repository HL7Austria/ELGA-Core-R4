# FHIR® ELGA Core Implementation Guide (R4) DRAFT

Dieser Implementierungsleitfaden beschreibt den **FHIR ELGA Core** und dient als gemeinsame Grundlage für die Nutzung von HL7® FHIR® Release 4 (R4) in der österreichischen **ELGA-Infrastruktur**.

Der Leitfaden definiert die allgemeinen Festlegungen für FHIR-basierte ELGA-Implementierungsleitfäden. Er enthält grundlegende Profile, Erweiterungen und weitere wiederverwendbare Artefakte, die unabhängig von einem konkreten Anwendungsfall verwendet oder von fachspezifischen Implementierungsleitfäden als Abhängigkeit eingebunden werden können.

Fachspezifische Anforderungen und Anwendungsfälle sind nicht Bestandteil dieses Leitfadens. Diese werden in darauf aufbauenden Implementierungsleitfäden für die jeweiligen ELGA-Anwendungen definiert.
